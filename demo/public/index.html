  • Type toFountain() in the console to output the screenplay in Fountain.
  • Type fromFountain(`...`) in the console to import a Fountain screenplay.
  • Keybinds:
    • TAB to cycle between node types.
    • Mod+I for italic, Mod+B for bold, Mod+U for underline.
    • Mod+D to insert or convert to a dual dialogue.
  • Shortcuts:
    • Type "INT.", "EXT.", or "." to create a scene heading.
    • "@" for a character
    • ">" or "TO:" for a transition
    • "=" for a synopsis.
    • "*text*" for italic, "**text**" for bold, "_text_" for underline.
